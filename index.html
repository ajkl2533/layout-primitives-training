LAYOUT PRIMITIVES

What are layout primitives?

Basic and simple building blocks
meant to be used in composition
to create bigger layouts

Based on Every Layout
by Heydon Pickering and Andy Bell

Basic layout primitives rules

  1. Primitive have only one responsibility
  2. Composition creates complex layouts
  3. No business logic in primitives
  4. No custom styling on primitives
